Actividad final

Deberán realizar un sistema de administración de usuarios; este sistema deberá contener un encabezado y un menu como el mostrado en la imagen.



Y deberá tener las siguientes características:


  1. Cada sección del menu debe ser funcional.
  2. Se deberá programar con el patrón modelo-vista-controlador.
  3. Deberá tener un acceso a base de datos cuya nombre sera escuela; una tabla cuyo nombre sera usuarios y cuatro campos: id, nombre, password y email. Esta conexión deberá realizarse mediante una clase.
  4. El menu Inicio deberá llevarte a una pagina de inicio. (libre para diseñar)
  5. El menu Insertar deberá mostrar una pantalla para insertar los datos.

  1. El menu Accesar deberá Accesar una pantalla de login para accesar datos al sistema.

  1. El menu Ver deberá mostrar los usuario que están en la base de datos de usuarios con el siguiente formato:

  1. El botón editar deberá llevarnos una pantalla en donde se verán los dato a editar. Después de actualizar datos nos deberá regresar a la pantalla de Ver usuarios para comprobar que se realizó la actualización

  1. El botón borrar deberá eliminar al usuario en el que se encuentra el botón.Después de actualizar datos nos deberá regresar a la pantalla de Ver usuarios para comprobar que se realizó el borrado del usuario.
  2. El menu Salir nos sacara del sistema


Se hace notar que para poder realizar cualquier modificación en el sistema primero se deberá accesar con usuario y contraseña. Debido a esto es conveniente que exista por lo menos un usuario registrado en la base de datos.
